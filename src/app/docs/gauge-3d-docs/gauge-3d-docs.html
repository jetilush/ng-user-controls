3D Gauge

An animated speedometer-style dial with threshold color zones, a lerp-animated needle, tick marks, and an HTML value overlay — all rendered in Three.js.

Three.js Animated Needle Threshold Zones Bezel + Glass Orbit Controls

How to use

Bind [value] to any numeric signal. The needle lerps smoothly to the new position. Define thresholds in ascending order — each entry colors from the previous threshold (or min) up to its value.

Design

  • 240° arc sweep (7 o'clock → 5 o'clock)
  • 80 colored box segments per arc
  • Needle lerps to target value at 7 % / frame
  • Gauge body: CylinderGeometry face + TorusGeometry bezel
  • Glass cover (MeshPhongMaterial, opacity 6 %)
  • Value display: HTML overlay (no canvas texture redraws)
  • 5 tick marks with canvas-sprite labels
  • Threshold zones configurable — supports reversed polarity
  • Drag to orbit, scroll to zoom

Reversed polarity

For "higher is better" metrics, sort thresholds red → yellow → green:

// Health score: 0–30 bad, 70–100 good
[
  { value: 30,  color: '#ef4444' },
  { value: 70,  color: '#eab308' },
  { value: 100, color: '#22c55e' }
]
