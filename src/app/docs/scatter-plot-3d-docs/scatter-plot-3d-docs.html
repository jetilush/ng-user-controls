3D Scatter Plot

Plots arbitrary point clouds in true 3D space using InstancedMesh — fast, colorable by series, with hover inspection and click selection.

Three.js InstancedMesh X / Y / Z Axes Series Legend Hover + Click

How to use

Pass a points array of { x, y, z, color?, series?, size?, label? }. If series is set, points are coloured from a built-in palette (or per-point color overrides it).

Point interface

interface ScatterPoint {
  x: number;
  y: number;
  z: number;
  label?: string;
  color?: string;  // hex / rgb
  series?: string; // auto-colored
  size?: number;   // size multiplier
}
