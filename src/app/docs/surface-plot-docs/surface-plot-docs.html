3D Surface Plot

Renders a 2D grid of values as a height-mapped mesh with vertex coloring, animated rise, and live hover inspection.

Three.js Vertex Colors Animated Hover Sphere Color Scales

How to use

Pass a data input of type number[][] — a 2D grid where each cell is a scalar value. The component normalises the range and maps it to the chosen color scale.
